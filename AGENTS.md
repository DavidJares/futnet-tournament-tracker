# FTT MVP rules

Project goal:
Build a lightweight tournament management web app for small local tournaments.

Tech stack:
PHP 8.x
MySQL
Server rendered pages
Vanilla JS only when necessary
Bootstrap allowed

Constraints:
Do not introduce Node.js build tools
Do not use websockets
Do not add unnecessary abstractions
Prefer simple readable PHP
Prefer mobile friendly admin flows
Keep hosting compatibility with standard shared hosting

Coding rules:
Use clear folder structure
Keep controllers thin
Put SQL access in models
Use prepared statements
Do not hardcode credentials
Document each new feature in README.md
